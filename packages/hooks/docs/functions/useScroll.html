      控制页面或元素的垂直滚动，提供滚动到顶部/底部的功能

      可用于实现无限滚动（当isAtBottom为true时加载更多数据）或返回顶部按钮等功能

      Parameters

      • options: { isWindow?: boolean; smooth?: boolean; threshold?: number } = {}

        配置选项

        • OptionalisWindow?: boolean

          是否监听window滚动，默认false

        • Optionalsmooth?: boolean

          是否使用平滑滚动，默认true

        • Optionalthreshold?: number

          距离底部的阈值（像素），默认450px

      Returns {
          goToBottom: () => void;
          goToTop: () => void;
          handleScroll: () => void;
          isAtBottom: boolean;
          ref: RefObject<Window | HTMLElement | null>;
      }

      返回包含滚动状态和控制函数的对象

      Example

      function ScrollComponent() {
        const { isAtBottom, goToTop, goToBottom } = useScroll({
          isWindow: true,
          threshold: 500,
        });

        useEffect(() => {
          if (isAtBottom) {
            // 加载更多数据
          }
        }, [isAtBottom]);

        return (
          <div>
            <button onClick={goToTop}>返回顶部</button>
            {isAtBottom && <div>Loading more...</div>}
          </div>
        );
      }
